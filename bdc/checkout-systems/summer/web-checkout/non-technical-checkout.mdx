---

The Summer Checkout flow includes:

1. Login / Sign-up via cart page
1. Entering student details
2. Selecting a session and location
3. Optionally choosing future bundle programs
4. Completing payment

Each step must be completed before moving forward.

## Step 1: Login / Sign-up via cart page

This step work same as the  semester page login and signup sections

## Step 2: Student Details

### Information Collected

This is the first step of the checkout process . It's the same as the semester student details form.

The user enters the studentâ€™s basic information, including first name, last name, email address, grade, school name, gender, and whether the student has previously participated in a BDC class.\
After completing the form, clicking **Next** saves the information and moves the user to the session selection step.

## Step 3: Select Session and Location

In this step, the user selects when and where the summer program will take place. 

The user first chooses a session (for example, Session 1 or Session 2). Based on the selected session, available locations are shown. The user selects one location and clicks **Next**, which opens the bundle program popup.

## Step 4: Bundle Program Popup (Optional)

This optional step allows users to add future programs as a discounted bundle. 

A popup displays available bundle programs. The user can select one or more bundle programs or skip this step by clicking **No Thanks**. Closing the popup moves the user to the payment screen.

## Step 5: Payment Screen

![Image](/images/image-21.png)

This is the final step where the user reviews details and completes payment. Once checkout details are confirmed, payment options appear. Available payment methods include **bank transfer**  and **credit card**. After selecting a payment method, the **Buy Now** button becomes available.\
Clicking **Buy Now** redirects the user to the Stripe Checkout page.

### Bundle Program Banner

A Bundle Program Banner is visible on the payment screen. It shows any selected future bundle programs and allows users to review, add, or update bundle selections directly from the banner.

### Order Details

The order details section displays a full summary of the registration. It includes the selected summer program and session, selected location, selected bundle programs, and a price breakdown with the total amount.

## 6. Stripe Checkout

**On the Stripe Page: **the user enters payment details securely. Stripe processes the payment, and no payment information is stored on the website

**After Successful Payment: **After payment is completed, the user is redirected to a success or confirmation page. The student is successfully registered for the selected summer program.
