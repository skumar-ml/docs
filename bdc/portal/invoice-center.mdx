This page allows admins to generate and manage invoices for students for a selected session and year. Admins first filter and select students, then choose how the payment will be collected.

Payment Methods

JotForm: Used when payment needs to be collected via an external JotForm link. Admin adds the form URL and sends it to parents.

Stripe: Used for direct online payments. Admin sets card and/or bank transfer amounts that parents can pay through Stripe.

Bucket: Used when the invoice amount is adjusted using internal credits or predefined amounts (no immediate online checkout).

Invoice Setup

Admin can apply returner, sibling, or early-bird options if applicable.

Different amounts can be set for card vs bank transfer.

If an invoice already exists, it can be edited instead of recreated."
